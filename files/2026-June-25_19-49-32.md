Why is DNS Important?Enables accessibility of websites by allowing people to use names instead of complex numbers.Supports a distributed, scalable internet architecture.How DNS Lookup WorksAs shown in the graphic on the slide:The Request: A User queries the network for the IP address belonging to a domain name (e.g., "What is the IP address for Google.com?").The Resolution: The request goes to a DNS Server, which looks up its database and returns the corresponding IP address (e.g., 74.125.142.147) back to the user.Architecture Mindset: As system architects, it is vital to treat DNS as the absolute first step in almost every network request.If you are studying network fundamentals, let me know if you want to explore how DNS caching works, the differences between types of DNS records (like A, AAAA, or CNAME), or how to run a live lookup using your computer's command terminal!
